# NoteHub

NoteHub is a simple notes app for storing, creating, deleting, and searching notes. It provides a clean UI and fast data updates.

## Features

- ✅ Create notes (title, content, tag)
- ✅ Delete notes
- ✅ Search notes by keyword
- ✅ Filter/group notes by tag (optional, if implemented)
- ✅ Form validation (Formik + Yup)
- ✅ Instant UI updates with React Query

## Tech Stack

- React + TypeScript
- Formik + Yup (forms and validation)
- @tanstack/react-query (server state management)
- CSS Modules (styling)
- API service layer (`noteService`)

## Getting Started

### 1) Install dependencies

```bash
npm install
```
